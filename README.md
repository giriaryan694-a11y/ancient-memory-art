# Avadhān Vidyā — Ancient Memory Art

[Live Demo](https://giriaryan694-a11y.github.io/ancient-memory-art/)

## Description

Avadhān Vidyā is the ancient Indian art of supreme concentration, memory, and multitasking. This project presents a **21-day beginner course** to train your focus, memory, and creative recall. The webpage is fully responsive, making it easy to follow on **mobile phones, tablets, and desktops**.

## Features

* **21-Day Beginner Course:** Step-by-step exercises to build concentration, memory, and multitasking.
* **Daily Routine Examples:** Pranayama, Trataka, Peg/Link Story, Creative Avadhān.
* **Tools & Techniques:** Memory palace, peg system, linking stories, visual number-images.
* **Theme Toggle:** Switch between dark and light mode; your preference is saved in cookies.
* **Responsive Design:** Works seamlessly on phones and desktops.

## Why Learn Avadhān Vidyā?

* Improves focus and attention span.
* Enhances memory and recall abilities.
* Teaches multi-tasking and creative thinking.
* Revives a centuries-old discipline practiced by scholars, monks, and poets.

## How to Use

1. Open the [live demo](https://giriaryan694-a11y.github.io/ancient-memory-art/) in your browser.
2. Follow the 21-day course day-by-day.
3. Practice daily routines and exercises.
4. Toggle between dark/light mode as per preference.
